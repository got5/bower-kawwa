Kawwa Example Templates

Default Theme / Version 1

Forewords

In this pack you will find some templates that may be used as starting points for your own pages. Feel free to modify them as you want!

With this pack you have:

  • 8 HTML files, in the "templates" folder, each one of them present examples of components for a specific category.
  • Example images and icons, in the "img" folder: feel free to use and modify them
  • 3 style-sheets, in the "css" folder:
    • "k-structure.css": resets the styles and defines the main page structure
    • "k-theme0.css": main styles for the theme
    • "i-hacks0.css": fixes for Internet Explorer, versions prior to version 9

Your feedback is welcome! Send us your comments and suggestions to our mailing list dl-kawwa@atos.net.

Licenses

Web Fonts

Kawwa's default theme uses two web font families:

  • Novecento Wide: this font is used for titles and navigation
  • PT Sans: this font is used for all body texts. Refer to its License file in the "css" folder.

Because of its End-User License Agreement, EULA, the web font family Novecento Wide cannot be bundled ready to use with the templates. If you want to use this web font family, you can download it for free and ready to use at FontSpring.

Once downloaded the web font files, all you have to do is to add those files to your "css" folder. Font stacks are already defined for those fonts in your style-sheet.

Before using web fonts, please check their EULA.
